# Plantilla estándar de publicación - LaCajaAmbar

## Naming recomendado
`YYYY-MM-tema-slug-###`

Ejemplo:
`2026-05-propoleo-mitos-002`

## Estructura de carpeta
- `bruto/` → vídeos, fotos, audios y notas de origen
- `produccion/` → guion, voz, subtítulos, copies, portada provisional
- `final/` → mp4 final, caption final, portada final, checklist

## Archivos mínimos por pieza
- `briefing.md`
- `guion.md`
- `caption.txt`
- `textos-pantalla.md`
- `checklist-publicacion.md`

## Estado operativo sugerido
- idea
- en-guion
- en-produccion
- listo-para-publicar
- programado
- publicado

## Checklist rápida
1. Tema claro
2. Formato definido
3. Hook inicial fuerte
4. Texto en pantalla revisado
5. Caption final listo
6. CTA si aplica
7. Portada definida
8. Asset final exportado
9. Programación en Metricool
10. Estado actualizado
