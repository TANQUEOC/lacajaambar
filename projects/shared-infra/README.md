# Shared Infra

## Identidad
- Nombre: Shared Infra
- Slug: shared-infra
- Tipo: infraestructura
- Estado: activo

## Objetivo
Centralizar piezas técnicas reutilizables entre varios proyectos TECH.

## Resultado esperado
Tener una zona clara para utilidades comunes, conectores, helpers y patrones compartidos.

## Reglas operativas
- Solo van aquí piezas realmente compartidas.
- Nada específico de un cliente salvo si se generaliza primero.
