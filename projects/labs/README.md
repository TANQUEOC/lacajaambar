# Labs

## Identidad
- Nombre: Labs
- Slug: labs
- Tipo: laboratorio
- Estado: activo

## Objetivo
Guardar pruebas, experimentos e ideas que aún no merecen un proyecto formal separado.

## Resultado esperado
Reducir el caos de pruebas sueltas y facilitar que una idea madura se convierta luego en proyecto real.

## Reglas operativas
- Aquí sí caben pruebas.
- Si algo empieza a tener continuidad, se saca a su propia carpeta de proyecto.
