# LaCajaAmbar · Identidad de marca extraída de la web

## Objetivo
Dejar una base de identidad usable para cualquier contenido, diseño, prompt, documento o material comercial de LaCajaAmbar.

## Fuente
- web analizada: `https://lacajaambar.com/`
- revisión basada en HTML, CSS y recursos públicos visibles

## Resultado
Se ha detectado una identidad de marca con estos rasgos principales:
- natural
- orgánica
- serena
- limpia
- visualmente aireada
- vinculada a abeja, biodiversidad, paisaje y productos de la colmena

## Uso recomendado
### Para contenido
- tono cercano, humano y riguroso
- belleza sin artificio
- divulgación suave
- CTA discretos

### Para diseño
- pocas familias tipográficas
- fondos claros o azul profundo
- fotografía natural real
- composición limpia
- nada estridente
